# AI News Frontend

- React + Vite + Tailwind
- Темы: светлая/темная
- Страницы: Главная (список), Детальная, Настройки
- Интеграции с вебхуками через Axios, заголовок Authorization установлен

## Запуск

1. Установите зависимости:
```bash
npm install
```
2. Запуск разработки:
```bash
npm run dev
```

Настройки темы сохраняются в localStorage (`theme`).
